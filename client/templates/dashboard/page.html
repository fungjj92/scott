How to use this site

Scott Eustis and Thomas Levine are the main persons who enter data into the site. Everyone can view this entire site and see how the process works, but you need to be logged in to enter data.

Changes to the data are sent to the server right away. If you aren't sure whether something has been saved, reload the page and check that it matches your expectations.
